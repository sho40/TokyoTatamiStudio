機能（今後の実装内容含む）
* pay.jpを利用した購入機能
* sessionを利用したカート機能実装

* UIを意識したフロントサイドの実装 　 (TOPページのヘッダーにて現在の時間に合わせてopenしているかcloseしているかを表示など)
